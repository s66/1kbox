1kbox api

  • $1k

    • $1k.console(msg) 向控制台输出信息
    • $1k.alert(msg) 弹出信息

  • Fs.js

    • Fs.codepage(codepage) 设置Fs默认编码
    • Fs.read(path) 读取文件，path为文件的物理绝对路径
    • Fs.readText(path, codepage) 读取文本文件
    • Fs.writeText(path, content, codepage) 写文本文件
    • Fs.exists(path) 判断文件是否存在
    • Fs.delete(path) 删除文件
    • Fs.createFolder(path) 创建文件夹
    • Fs.getFolder(path) 获取文件夹
    • Fs.getFile(path) 获取文件
    • Fs.getExt(path) 获取文件扩展名
    • Fs.syncText(targetPath, text, codepage) 同步文本文件
    • Fs.syncFile(targetPath, sourcePath) 同步文件
    • Fs.syncFolder(targetPath, sourcePath, filter) 根据过滤器同步文件 
       filter为过滤器，过滤条件可以是文件或者文件夹名称
            filter = {
                matchFile: /\.js$/, //匹配以.js结尾的文件
                skipFile: /\.res\.js$/, //跳过以.res.js结尾的文件
                matchFolder: /.+/, //匹配所有文件夹
                skipFolder: /\.res$/ //跳过以.res结尾的文件夹
            };
    • Fs.eachFiles(from[, filter][, deep], fn) 遍历操作指定文件夹中的文件,filter,deep默认false

  • Packer.js

    • Packer.pack(jscode) 压缩js代码

  • Zero.js

    • Zero.codepage(codepage) 设置模版引擎默认编码
    • Zero.getContent(path) 调用模版引擎生成html代码

    模版引擎扩展语法

    • $PATH 模版引擎当前操作的文件路径
    • $inc(path) 相对当前文件导入其他文件
    • $require(path) 相对当前文件加载js模块
    • $mappath(path) 获取文件的绝对物理路径
    • $push(text) 插入字符串
    • $each(items, func) 遍历操作items

  • Response.js

    • Response.setHeader(key, val) 添加响应头
    • Response.setBuffer(size) 设置缓存大小
    • Response.flush() 立即输出缓存内容
    • Response.end() 结束输出
    • Response.write(text) 输出字符串内容
    • Response.BinaryWrite(content) 输出2进制内容

  • Sys.js

    • Sys.wait(ms) 服务器进入等待，单位为毫秒
    • Sys.loop(fn, ms, count) 服务器循环执行指定的代码，fn为执行的函数，ms为暂停的间隔，count为执行次数

  • 其他

    • $HTTPPATH 当前网站域名
    • $ROOTPATH 当前网站根目录
    • $CODEPAGE 当前网站编码
    • $GET 客户端get方式发送的数据
